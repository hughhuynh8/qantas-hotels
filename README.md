## How to install

### Run Locally

Clone the project

```bash
  git clone https://github.com/hughhuynh8/qantas-hotels
```

Go to the project directory

```bash
  cd qantas-hotels
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


### Initialization
**The default port is 3000**

Open http://localhost:3000 through the browser.


### Test

Run the tests

```bash
  npm run test
```
